Project: Smoothed-Particle Hydrodynamics simulation of Toy Star

Run

To run the different implementations in C++. Run the run.shbash sricpt with:

run.sh

Constants and other simulation parameters can be changed in constants.hpp. Data from the simulation is then stored in ./output/*

In the tools directory it is possible to both plot the results and validate compared to the original Python version. Make sure that the constants match in constants.hppand Python script sph.py. Also note that sph.py is dependent on existing output from the C++ implementation for initialization.

For results validation:

pip install -r requirements.txt
cd tools
python sph.py #Run python script
python compare_output.py

Profiling

To get profiling information from Craypat: Run the scripts in the profiling directory, e.g.

cd profiling
sbatch profiling_MPI_single.sh
python plot_profiling.py
