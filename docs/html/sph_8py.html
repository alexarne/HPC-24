54 """
55 Get pairwise desprations between 2 sets of coordinates
56 ri is an M x 3 matrix of positions
57 rj is an N x 3 matrix of positions
58 dx, dy, dz are M x N matrices of separations
59 """
60
61 M = ri.shape[0]
62 N = rj.shape[0]

65 rix = ri[:,0].reshape((M,1))

35 Gradient of the Gausssian Smoothing kernel (3D)

39 h is the smoothing length
40 wx, wy, wz is the evaluated gradient
